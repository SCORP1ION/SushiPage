# Sushi Landing Page 🍣

A modern and responsive landing page for a sushi restaurant built with Next.js and TypeScript.

## Features

- Responsive design
- Modern UI/UX
- Featured sushi menu
- Restaurant information
- Image gallery
- Reusable components
- CSS Modules styling

## Tech Stack

- Next.js
- TypeScript
- CSS Modules
- Git & GitHub

## Getting Started

```bash
pnpm install
pnpm run dev
